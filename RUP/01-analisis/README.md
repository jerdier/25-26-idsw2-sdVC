## Actores del Sistema
1. **Secretaría**: Gestión de alumnos y matrículas.
2. **Profesor**: Registro de asistencia.
3. **Alumno**: Solicitud de dispensas.
4. **Director de Grado**: Validación académica y aprobación de dispensas.
