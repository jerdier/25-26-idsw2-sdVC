# Imágenes

[← Inicio](../README.md) · [Modelos UML](../modelosUML/README.md) · [Documentos](../documents/README.md)

SVGs exportados de los diagramas PlantUML, organizados por fase y caso de uso.  
Los SVGs se generan a partir de las fuentes `.puml` en [`/modelosUML`](../modelosUML/README.md).

**Formatos:** `.svg` para diagramas UML · `.png` para capturas e imágenes de interfaz
